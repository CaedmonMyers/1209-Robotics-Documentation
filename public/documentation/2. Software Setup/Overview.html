Setting up Software

To deploy to the robot or simulate code, you have to use WPILib VS Code. You should download the newest version from the GitHub releases page below.

Supported Operating Systems:

  • Windows 10, 64 bit and Windows 11. Arm is not supported.
  • macOS 11 or higher for Intel (x86) and M-series (Arm)
  • Ubuntu 22.04, 64 bit
Download

You will also need to download the FRC game tools to use Driver Station to control the robot. The official version is only available for Windows.
