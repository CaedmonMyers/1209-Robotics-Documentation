Xbox Controller

To start using Xbox controllers in your code, you have to import the CommandXboxController library from WPILib. 

import edu.wpi.first.wpilibj2.command.button.CommandXboxController;

There is also a library available called XboxController. This is an older version of CommandXboxController and has mostly been replaced.

Before using the controller in your code, you have to create the CommandXboxController object. If you use multiple controllers, each one will need its own object and each object will need to have a different port (the number in parentheses). 

CommandXboxController joystick = new CommandXboxController(0);

You can use the controller in your code by calling a specific button and using the .onTrue and .whiteTrue functions. 

joystick2
    .leftBumper()
    .onTrue(
        Commands.run(
            () -> {
            // action here
            }
        )
    )
    .whileTrue(
        Commands.run(
            () -> {
                // action here
            }
        )
    );

This code only runs a single command on each action. For more advanced commands, see the command page in the documentation.
