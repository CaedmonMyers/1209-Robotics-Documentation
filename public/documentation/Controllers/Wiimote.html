Wiimote

You can control your robot using a Wiimote to an extent. To connect a Wiimote to your computer, you will need to use a bluetooth adapter. I've only tested it with the Mayflash Dolphinbar so far.

The controls for the Wiimote are set up using Generic HID. You can download the Java file set up for the controls below.

Download

You can then create the Wiimote controller object in your code like below. 

WiiRemote wiimote = new WiiRemote(0);

The Wiimote has to be used slightly differently than Xbox or PlayStation controllers for commands. You have to create a JoystickButton object before using commands with the controller. The JoystickButton object takes in 2 parameters. The first is the wiimote controller object. The second is the port of the button from the Wiimote. You also will have to import the WiiRemote code from the file you downloaded first. 

import frc.robot.utilities.WiiRemote;
new JoystickButton(wiimote, 1)
    .onTrue(
        Commands.run(
            () -> {
                wiiRemoteForward = true;
            }
        )
    )
    .whileTrue(
        Commands.run(
            () -> {
                wiiRemoteForward = true;
            }
        )
    )
    .onFalse(
        Commands.run(
            () -> {
                wiiRemoteForward = false;
            }
        )
    );

Unfortunately, this code for the Wiimote does not work with the accelerometer or gyroscope which means you cannot use any motion controls other than the infrared pointing control.
